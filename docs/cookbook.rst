:meth:`gTools.d_stamp`
----------------------
Call function to return date string, format show like: 2023-07-20 14:30:26
呼叫後回傳日期字串, 字串格式: 2023-07-20 14:30:26

.. code-block:: python

    from sitTools import gTools
    
    gTools.d_stamp()
    
Result:
執行結果
::
    2023-07-25 23:00:37
